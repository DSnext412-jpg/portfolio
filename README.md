# Premium Portfolio

A premium Next-Generation Developer Portfolio Website built with modern web technologies.

## Technology Stack
- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- GSAP
- React Three Fiber
- Three.js
- Various UI libraries

## Scripts

### Available commands

- 
pm run dev - Start the development server
- 
pm run build - Build for production (includes type checking)
- 
pm run build:prod - Build for production with optimizations
- 
pm run lint - Run linting
- 
pm run typecheck - TypeScript type checking
- 
pm run test - Run tests
- 
pm run analyze - Bundle analyzer
- 
pm run clean - Clean build artifacts

## Frontend

This is a frontend-only portfolio website that showcases my skills, projects, and professional journey.

All features are implemented using only front-end technologies, ensuring a seamless and integrated development experience.

No backend or database required; everything runs entirely in the browser.

### Key Features

**Core Sections**
- Hero Section
- About Me
- Skills (Programming, Database, AI/ML, Tools)
- Education
- Internship
- Experience
- Projects
- Certificates
- GitHub Dashboard
- Coding Profiles (LeetCode, GitHub)
- Statistics
- Contact (EmailJS form)
- Footer

**Interactive Elements**
- 3D Background (React Three Fiber, Three.js)
- Smooth animations (Framer Motion, GSAP)
- Theme switching (Light/Dark)
- Magnetic buttons
- Scroll progress
- Scroll to top
- Loading screen
- Mouse cursor effect
- Lazy loading
- Optimized bundle size

**Premium Features**
- Glassmorphism
- Gradient borders
- Responsive design
- SEO optimized
- PWA ready
- Lorem ipsum integration

## Building

1. Install dependencies: 
pm install
2. Run development: 
pm run dev
3. Build production: 
pm run build

## Deploy

Deploy to GitHub Pages or any static site hosting service.

The built output (dist/ folder) is ready for deployment.
